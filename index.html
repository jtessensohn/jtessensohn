Things I Have Made

Below are various projects that mark large milestones in my journey of being a developer. If the project is published, keep in mind that it may take a few seconds for server to spool up

applik8

applik8 was the first full-stack project I had a part in creating. As a member of a four person team, we built a website that enabled users to track their applications they had active through user-input, complete with dates of application and a note field. Users also had access to salary projections based on data from the Bureau of Labor Statistics, Glassdoor, etc. and used an averaging formula to spread our data across 15 years which would then be displayed on graphs.

Primary responsibilities included:
  • Login service w/ encryption & sessions
  • Database setup and management
  • Backend routing
Technologies include:
  • Node.js
  • Express
  • PostgreSQL & Sequelize

Destiny Item Viewer

Destiny Item Viewer is an application that currently allows a user to check stats of their/others Destiny 2 Characters. Utilizing multiple API endpoints and ordering of API calls, data is organized and utilized to return information cleanly and effectively. This project is far from finished and expect to be updating it as a pet-project in the future.

Technologies include:
  • React
  • Context
  • Tailwind
